В файле resources\views\components\page-header.blade.php сделай верстку секции согласно следующему описанию:
    внутри секции есть контейнер - простой div с классом container (он уже стилизован его не нужно стилизовать в рамках выполнения задачи)
    - У секции должен быть инлайн стиль в котором прописанно свойство background-image значение которого это url взятый из переменной $banner
    - У секции должен быть оверлей черный с прозрачностью 60%ж
    - Внутри контейнера расположен заголовок h1 белого цвета размер шрфта 48px заголовок прижат к левому краю.

Стили реализуй в scss, размести их в файле _page-header.scss. В написанном файле стилей используй переменные которые есть в файл _vars.scss, другие пременные не используй. Сделай адаптацию под устройства.
